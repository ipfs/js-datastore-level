Intro

Installable via npm install --save datastore-level, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save datastore-level

You can then require() datastore-level as normal:

const datastoreLevel = require('datastore-level')

In the Browser

Datastore-level should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of datastore-level, including index.js and index.min.js, are included in the /dist folder. Datastore-level can also be found on unpkg.com under

  • https://unpkg.com/datastore-level/dist/index.min.js
  • https://unpkg.com/datastore-level/dist/index.js
